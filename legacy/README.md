# WOX-Bin Legacy Project

This folder contains the original WOX-Bin implementation that existed before the Next.js/Vercel rebuild.

Contents:

- `index.html`, `styles.css`, `app.js`
- `js/` browser script bundle
- `server/` Express + sql.js backend

Legacy launchers:

- `run-local.bat` for the local-only static app
- `run-server.bat` for the old account-backed server app

The active Vercel-targeted app now lives in the repo root.
